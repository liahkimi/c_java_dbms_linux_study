변수
	값을 저장하는 저장공간
	값을 재사용 하기 위해서 사용한다.

변수의 선언
	자료형 변수명;
	ex) int num;

변수의 초기화
	선언된 변수에 최초의 값을 대입하는 행위
	선언된 변수 타입에 알맞은 값만 대입해 주어야 한다.
	
	변수명 = 값;

변수와 값 구분하기
	변수 이름 오른쪽에 대입 연산자가 있다면! 저장공간으로 쓰인다.
	변수 이름 앞에 자료형이 붙는다면 저장공간으로 쓰인다.
	대입 연산자가 없다면 ★★★값★★★ 으로 쓰인다.

소스파일 빌드에서 제외하는 방법
	해당 파일 마우스 우클릭 > 속성 > 일반 > 빌드에서 제외 > 예 > 적용 > 확인
	구성 : 활성(Debug)  플랫폼 : x64

제어문자 (이스케이프 문자)
	문자열 안에서 특수한 기능을 하는데 사용되는 문자
	
	\n : 줄바꿈, 엔터를 표현한다.
	\t : tab
	\" : 큰 따옴표
	\' : 작은 따옴표
	\\ : 역슬래쉬

서식지정자 (형식지정자)
	%d	int
	%c	char
	%lf	double

변수명 작성시 주의사항
	1. 이미 선언한 변수는 같은 이름으로 선언이 불가능하다.
	2. 대소문자를 구분한다.
	3. 띄어쓰기가 불가능하다.
	4. 특수문자는 사용할 수 없다. ('_', '$' 만 사용가능)
	5. 숫자로 시작할 수 없다.
	6. 키워드(예약어)는 사용할 수 없다.

& -> 주소 연산자
	변수의 주소값을 가져온다.
	&num -> num의 주소값

	주소값에 해당하는 형식 지정자 %d , %p
