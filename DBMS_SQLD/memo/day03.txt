day03. 연결 연산자, 비교연산자, 논리연산자, SQL 연산자

연결연산자
	문자 연결에 사용되는 연산자
	a || b : |(shift + 역슬래시) 
	a와 b 문자갑을 하나의 문자로 연결한다 

	* 숫자, 날짜, 문자 타입의 값에 사용이 가능하다

WHERE절
	조건에 만족하는 데이터를 조회하거나 연산한다
	* SELECT 는 조건식을 만족하는 행들만 조회한다
	-> 조건식의 실행결과가 참인 행들만 조회한다
	-> 행을 필터링 한다

	SELECT 컬럼명1, 컬럼명2...
	FROM 테이블명
	WHERE 조건식;

표현식
	값으로 평가될 수 있는 식(컬럼의 값, 하나의 값, 함수의 사용,
	연산자의 사용)
	* 조건식 : 표현식의 값이 TRUE, FLASE의 값이 나오는식

비교연산자(관계연산자)
	a < b : a가 b보다 작다(미만)
	a > b : a가 b보다 크다(초과)
	a <= b : a가 b보다 작거나 같다(이하)
	a >= b : a가 b보다 크거나 같다(이상)
	a = b : a와 b가 같다
	a <> b, a != b, a ^= b : a와 b가 같지 않다(다르다)

논리연산자
	1. AND(그리고) : a AND b
		a 조건식과 b 조건식 둘다 참이라면 참
		둘중 하나라도 거짓이라면 거짓

	2. OR(또는) : a OR b
		a 조건식과 b 조건식 둘 중에 하나라도 참이면 참
		둘다 거짓이라면 거짓

	3. NOT(부정) : NOT a
		a 조건식이 거짓이라면 참으로,
		참이라면 거짓으로(반대로 변경한다)

BETWEEN a AND b
	a 이상 b 이하를 만족하면 참
	BETWEEN 숫자1 AND 숫자2 : 숫자 1 이상, 숫자2 이하이면
	참(숫자1 <= n <= 숫자2)

	BETWEEN 날짜1 AND 날짜2 : 시작날짜 날짜1로부터 종료날짜
	날짜2 사이의 날짜이면 참(BETWEEN 예전날짜 AND 최신날짜)

	* 두개의 값은 대소관계 비교가 진행되기때문에
	BETWEEN 작은값 AND 큰값으로 작성해야 한다

IN
	IN(값1, 값2, 값3....)
	표현식의 값이 소괄호 안의 값들중 하나라도 일치하면 참
